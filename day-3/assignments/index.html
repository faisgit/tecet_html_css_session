CSS Project Hub — Assignments

10 project-based assignments tailored for your 5-hour CSS course. Each assignment includes requirements and success criteria.

1. Personal Profile Card

Create a profile card with image, name, role, short bio and two action buttons.

Requirements: Flexbox centering, avatar with rounded corners, button hover effect, box-shadow, responsive width.

2. Responsive Pricing Table

Design 3 pricing plans (Basic / Pro / Enterprise). Highlight the recommended plan.

Requirements: Grid or Flex layout, featured card styling, responsive collapse to single column under 600px.

3. Modern Login Page

Build a centered login form with inputs, focus states and a submit button.

Requirements: Center using Flexbox, gradient background, accessible labels, input focus outline, responsive layout.

4. Portfolio Gallery (Image Grid)

Create a gallery of images that reflows across breakpoints and shows a zoom-on-hover effect.

Requirements: CSS Grid, gap spacing, hover transform & transition, 3→2→1 columns at desktop→tablet→mobile.

5. Landing Page Header + Hero

Design a header with nav and a hero section containing a headline, subtext and call-to-action button.

Requirements: Responsive navigation (stack or hamburger on small screens), hero alignment with image/text, accessible button states.

6. Animated Loader

Create a visually pleasing loader (spinner, bouncing dot, or bar) using CSS animations.

Requirements: Use @keyframes, center the loader, ensure smooth easing and infinite loop.

7. Product Card UI

Design a product card including product image, title, price and an Add to Cart button.

Requirements: Hover lift effect, subtle shadow, responsive stacking of elements for narrow screens.

8. Static Weather Dashboard UI

Create a dashboard with a large temperature card and 3–4 metric cards (humidity, wind, forecast).

Requirements: Grid or Flex layout, readable typography, cards with consistent sizing and responsive behavior.

9. Sidebar + Content Layout

Build a two-column layout with a sticky sidebar that becomes a top bar on mobile.

Requirements: CSS Grid template areas or Flex; sidebar sticky positioning; mobile-first responsive breakpoint.

10. Blog Post Page

Design a readable article page with title, hero image, body text and a right-side related posts sidebar.

Requirements: Comfortable line-length, typographic scale, responsive stacking of sidebar below content on small screens.

Submission Guidelines

  • Each assignment: single HTML file with embedded CSS (or separate CSS file).
  • Include comments at the top describing which features you implemented.
  • Optional: Add a short screenshot and notes for responsive behavior.
